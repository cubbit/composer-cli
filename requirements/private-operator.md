# Private Dashboard CLI

All the operations for the public CLI plus

- `cubbit operator create/new --name=<name> --email=<email> --password=<password>`: to create a new operator from CLI

## Distributor

- `cubbit distributor create/new --name=<name> --email=<email> --password=<password>`: to create a new distributor
- `cubbit distributor list/ls`: to list distributors
- `cubbit distributor remove/rm <distributor-id>`: to delete a distributor
- `cubbit distributor assign-tenant --tenant-id=<tenant-id> --coupon-id=<coupon-id>`: assigns a tenant to a coupon
